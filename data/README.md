# Data

This directory contains the dataset used for training the age prediction model.

## UTKFace Dataset

The dataset used is the [UTKFace large scale face dataset](https://susanqq.github.io/UTKFace/). It consists of over 20,000 face images with annotations for age, gender, and ethnicity.

The images in `data/dataset/UTKFace` have been pre-processed and are ready for use in the training notebooks.
